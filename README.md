# Treasure Grid — רואים דרך ה-AI

משחק לימודי: קירוב ליניארי + **TD** (טעות = פרס + γ×max(ציון-המשך) − ציון צעד), ε יורד עם הזמן, פיצ'רים כולל f₃ מובילות לאוצר ו-f₂ מרחק תמידי למלכודת.

## הרצה

דפדפן מקומי עם שרת סטטי (מודולי ES):

```bash
npx --yes serve .
```

אז פתחו את הכתובת שמוצגת (למשל `http://localhost:3000`) ופתחו את `index.html`.

אפשר גם **Live Server** ב-VS Code על תיקיית הפרויקט.

## קבצים

| קובץ | תפקיד |
|------|--------|
| `src/config.js` | קבועים (רשת, α, ε, תגיות) |
| `src/features.js` | פיצ'רים ∈ {-1,0,1} |
| `src/ai.js` | ציונים + ε-greedy |
| `src/learning.js` | עדכון משקלים |
| `src/game.js` | חוקים, פרסים, אוצר משותף |
| `src/render.js` | ציור לוח |
| `src/input.js` | מקלדת + כפתורים |
| `src/ui.js` | פאנלים והסברים |
| `src/main.js` | לולאה ו-200ms בין תורים |
| `src/errorToasts.js` | שגיאות מוצגות בטוסט (קובץ + שורה) |
| `favicon.svg` | אייקון לשונית ולמנועי חיפוש |

## שליטה

- **חצים** או **WASD** — כיוון
- **משחק מחדש** — לוח חדש + משקלי ברירת מחדל `(2, 1.5, 0.8, 0.25, 0.45)`
- **משקלי ה-AI** (מעל הלוח) — עריכה ידנית + **החל**, **חזרה לברירת מחדל**, **אפס ל-0**, או **משחק חדש + AI מאפס**
- **יומן למידה** בפאנל הצד — אחרי כל תור נוספת שורה (החדש למעלה), בלי לעצור את המשחק
- פאנלים **חישוב AI** / **יומן למידה** ניתנים לכיווץ בלחיצה על הכותרת
